## 🛠 Texnologiyalar

- **Frontend:** Next.js 15, TypeScript, Tailwind, Framer Motion
- **Backend:** FastAPI, Python, PostgreSQL, Redis, WebSocket
- **Mobile:** Kotlin, Android Compose
- **AI:** GroqCloud (Llama 3.3 70B)
- **Bot:** Telegraf
- **Cloud:** GitHub Pages, Cloudflare Workers, R2, D1
- **Cost:** $0/oy bepul
